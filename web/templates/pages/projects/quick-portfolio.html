I always have my nose in some kind of a project. Throughout the years, I've traversed so many spontaneous builds that I lost track. I wanted to create some kind of central repository where I could quickly type up my solutions to the irregular problems I face on the regular and larger projects I've worked on.

Welcome to the solution to that problem! RichThe.Dev is built on top of Laravel using the usual suspects: PHP, jQuery, Vue.js, SASS, and MySql.

After finding myself scouring the cscareerquestions subreddit, I noticed that an abundant amount of ambitious junior devs were eager to quickly create their own portfolio website. However, many of these devs have a focus on languages that aren't traditionally used for web development.

I decided to repurpose this site and package it on github to help these aspiring developers get their site up and running with no hassle.

Installation is easy: clone repo, cd into it from CLI, update .env with your creds, then run:
$ composer install
$ composer dumpautoload -o
$ php artisan key:generate
npm run dev
php artisan serve
And you're all ready to go!
